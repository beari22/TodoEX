# TodoEX

### installation
1. make sure you have python 3.10.x
1. do `git clone https://github.com/beari22/TodoEX.git`
1. do `cd TodoEX`
1. do `pip install pip install -r requirements.txt`
1. run app.py or `py cli_app.py {options} {args}`

### how to use 
do `py cli_app.py help`
